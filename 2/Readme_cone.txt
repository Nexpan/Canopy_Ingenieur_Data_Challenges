Pour exécuter le programme, ouvrez une fenêtre de terminal (ou invite de commande pour Windows), puis naviguez vers le répertoire où vous avez enregistré le fichier "cone_simulation.py".

Utilisez la commande suivante pour exécuter le programme en spécifiant le nombre (positif) de simulations ("-n") que vous souhaitez effectuer :
    python cone_simulation.py -n nombre de simulation
    ex: Nombre de simulation est éégal à 10
        python cone_simulation.py -n 10
   
Dans cette exemple, le programme générera une estimation du volume du cône basée sur les 10 simulations,
